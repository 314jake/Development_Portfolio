Mystery Organism

Simulates the findings of a laboratory working with a mystery organism named P. Aequor.

It generates a library of 30 specimens each with their own DNA sequence, and can manipulate this data in a few ways. Each specimen is a JS object with a number of methods.

In this example, one specimen's evolutionary line will be predicted as its DNA sequence mutates through generations and the script will find the two individuals in the library with the closest genetic match.
